# T42 IsaacLab Compat Decision

DECISION: DROP

## Basis

Phase 0 recorded `IsaacLab internals: 0` in
`.sisyphus/evidence/phase-0-assessment.md`. The Phase 0 scope rule says an
IsaacLab score of 1 or lower drops the Task 42 IsaacLab compatibility case
study entirely.

The same assessment also cut the standalone demo site, so the v0.1 deliverable
for this decision is documentation evidence, a roadmap update, and an honest
section in the generated docs site.

## v0.1 Scope

- No IsaacLab case study ships in v0.1.
- No general IsaacLab compatibility layer ships in v0.1.
- No USD conversion shim, IsaacLab parity claim, or reference-performance claim
  ships in v0.1.
- The v0.1 compatibility story remains MuJoCo tasks plus MCP tools.

## v0.2 Milestone

IsaacLab compatibility moves to a v0.2 research milestone. The first acceptable
v0.2 entry point is one measured task conversion with explicit divergence notes,
not a broad compatibility promise.
