Decision:
Use `physlab-mac` as the PyPI distribution name and `physlab-mac-mcp` as the
npm package name for v0.1. The Python import package remains `physlab`.

CI packaging fix:
The hello_task example package originally depended on the old `physlab`
distribution name. In CI, `uv pip install -e examples/plugins/hello_task`
resolved PyPI's unrelated `physlab` package and shadowed the local import
package. The example package now depends on `physlab-mac`, and
tests/integration/test_extension_freedom.py asserts this release distribution
name so the regression is caught before publish.
